⚡ Quick Start Commands

Install NgRx

npm install @ngrx/store@17 @ngrx/effects@17 @ngrx/entity@17 @ngrx/store-devtools@17

Generate Store

ng generate @ngrx/schematics:store --root

Generate Feature

ng generate @ngrx/schematics:feature --name=todos --reducers=index.ts
